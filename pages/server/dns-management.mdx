# Ganti DNS Permanent
Konfigurasi berikut adalah cara untuk mengganti DNS secara permanent, karena jika kita mengganti DNS dengan cara **sudo nano /etc/resolv.conf** maka ketika mesin server kita mati DNS secara otomatis akan balik ke DNS provider.

## Install Resolvconf
``` bash
sudo apt install resolvconf	
```
``` bash
sudo systemctl enable resolvconf
```
``` bash
sudo systemctl start resolvconf
```
``` bash
sudo systemctl status resolvconf	
```
``` bash
sudo resolvconf --enable-updates
```
Tambah konfigurasi di **/etc/resolvconf/resolv.conf.d/head** seperti berikut:
``` bash filename="/etc/resolvconf/resolv.conf.d/head"
nameservers 8.8.8.8
```

Jika sudah maka tinggal update **resolvconf** agar konfigurasi berjalan.
``` bash 
sudo resolvconf -u	
```
